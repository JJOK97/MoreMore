## FE

---

- 화면 구현(그룹 일정, 그룹 피드 화면, 상세 게시물 화면, 게시물 작성 화면, 그룹 선택 화면, 로그인 및 회원가입, 그룹 생성 화면, 그룹 정보 화면)

### 작업 내용
| 페이지 | 설명 |
| ------ | ------ |
| Login | 로그인 |
| Signup | 회원 가입 프로세스 |
| RegisterAccount | 개인 계좌 등록 |
| CreateGroup | 참여 그룹 생성 |
| GroupAccount | 그룹 계좌 화면 |
| Schedule | 그룹 일정 |
| Feed | 그룹 게시판 |
| Post | 그룹 게시판 상세보기 |
| GroupAccountDepositStatus | 그룹 계좌 입금 현황 |
| GroupDuesSetting | 그룹 계좌 회비 설정 |
| GroupInfo | 그룹 정보 |
| AccountTransfer | 계좌 송금 |
| GroupAccountSearch | 계좌 사용내역 검색 |
| TransactionDetail | 지출 세부 내역 |


### BE
- API 구현(ClubService, PostingService)
- 배포 구현 중(이번 주 내로 완료 예정)

| 서비스       | 메서드명          | 설명               | HTTP 메서드 | 엔드포인트                | 요청 변수 (Path/Request Body) |
|--------------|-------------------|--------------------|-------------|---------------------------|------------------------------|
| **CommentService** | `createComment`   | 댓글 생성           | POST        | /comments                 | Comment 객체                |
|              | `findByPostingId`   | 게시글 ID로 댓글 조회 | GET         | /comments/{postingId}      | `postingId` (Path)           |
|              | `deleteByCommentId` | 댓글 ID로 삭제       | DELETE      | /comments/{commentId}      | `commentId` (Path)           |
|              | `updateComment`     | 댓글 수정           | PUT         | /comments                 | Comment 객체                 |
| **LikeService**  | `isLike`          | 좋아요 여부 확인      | POST        | /likes/check              | LikeRequest 객체             |
|              | `likePost`         | 좋아요 추가         | POST        | /likes                    | LikeRequest 객체             |
|              | `unlikePost`       | 좋아요 취소         | DELETE      | /likes                    | LikeRequest 객체             |
|              | `getLikeCount`     | 게시글의 좋아요 수 조회| GET        | /likes/count/{postId}      | `postId` (Path)              |
|              | `getAllPosts`      | 좋아요 있는 모든 게시글 조회| GET    | /likes/posts/{postId}      | `postId` (Path)              |
|              | `getLikeCountsForPostings` | 여러 게시글의 좋아요 수 조회 | POST | /likes/counts             | List<Long> (Request Body)    |
|              | `deleteByPostId`   | 게시글 ID로 좋아요 삭제 | DELETE    | /likes/{postId}            | `postId` (Path)              |
| **PostingService** | `create`         | 게시글 생성           | POST        | /postings                 | Posting 객체, MultipartFile[] |
|              | `findByClubCode`    | 클럽 코드로 게시글 조회 | GET       | /postings/club/{clubCode}  | `clubCode` (Path)            |
|              | `findByPostId`      | 게시글 ID로 게시글 조회 | GET       | /postings/{postId}         | `postingId` (Path)           |
|              | `updatePostingById` | 게시글 ID로 수정      | PUT        | /postings/{postId}         | `postingId` (Path), PostingUpdateRequest 객체 |
|              | `PostingDeleteByPostId` | 게시글 ID로 삭제 | DELETE    | /postings/{postId}         | `postingId` (Path)           |
| **ClubService**   | `createClub`     | 클럽 생성           | POST        | /clubs                    | Club 객체, `creatorId`, MultipartFile |
|              | `updateClub`       | 클럽 정보 수정        | PUT         | /clubs/{clubCode}          | `clubCode` (Path), Club 객체 |
|              | `updateClubImage`  | 클럽 이미지 수정      | PUT         | /clubs/{clubCode}/image    | `clubCode` (Path), MultipartFile |
|              | `addParticipant`   | 클럽에 참가자 추가     | POST        | /clubs/{clubCode}/participants | `clubCode` (Path), List<Participant> 객체 |
|              | `findParticipants` | 클럽 참가자 조회      | GET         | /clubs/{clubCode}/participants | `clubCode` (Path)         |
|              | `findClub`         | 클럽 조회           | GET         | /clubs/{clubCode}          | `clubCode` (Path)            |
|              | `findClubs`        | 멤버 ID로 클럽 조회    | GET         | /clubs/member/{memberId}   | `memberId` (Path)            |
